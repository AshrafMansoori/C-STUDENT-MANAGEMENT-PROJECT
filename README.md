# Student Result Management System

This project is a simple C program for managing student records. It allows users to:

- Add student details
- Display all students
- Search for a student by roll number
- Save student data to a file

## How to Run

1. Compile the program:
   ```bash
   gcc PROJECT.C -o PROJECT.exe
   ```

2. Run the executable:
   ```bash
   PROJECT.exe
   ```

## Features

- Stores student roll number, name, and marks
- Calculates total, average, and grade
- Saves records to students.txt
